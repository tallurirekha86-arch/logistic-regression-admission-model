# Student Admission Prediction Model

A binary classification model built from scratch using Python and NumPy to predict university admission probabilities based on applicant metrics. This project was developed as part of the DeepLearning.AI Machine Learning Specialization by Andrew Ng.

##  Core Features
- **Vectorized Implementation:** Developed a logistic regression model using pure NumPy matrix operations instead of high-level machine learning frameworks.
- **Optimization Strategy:** Implemented gradient descent from scratch to optimize parameters by minimizing the binary cross-entropy cost function.
- **Regularization:** Integrated L2 regularization to minimize overfitting and improve the classification boundary across testing distributions.

## Tech Stack & Concepts
- **Languages:** Python (Jupyter Notebook)
- **Libraries:** NumPy, Matplotlib (for data visualization)
- **Mathematical Frameworks:** Cost Functions, Vectorization, Regularization, Binary Classification

## Performance Metrics
The model visualizes the decision boundary and calculates classification accuracy over the dataset parameters, showing precise engineering control over underlying algorithmic math.
